### Wokwi projekt
https://wokwi.com/projects/456388986903382017

Vi kompilerar med ```make``` i terminalen (Git Bash eller MinGW på Windows) för att få ut en körbar hex-fil.
I Wokwi klickar man var som helst i sketch-fönstret till vänster, trycker på ```F1```, söker efter ```upload```, väljer ```Upload Firmware and Start Simulation```, väljer hex-filen och trycker på ```Open``` så laddas hex-filen upp och startas automagiskt.
